CSS Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favorite CSS code? Select any snippet below and it'll automatically select all of the code for you to copy.

Flexbox Row

Use these three properties to create a Flexbox row layout.

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

Use this to create a Flexbox column layout.

.column {
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Build a X column layout using CSS Grid

.grid {
    display: grid;
    width: 100%;
    grid-template-columns: repeat(X, 1fr);
}

Linear Gradients

This will create a background gradient from top to bottom.

.linear-gradient-background {
    background-image: linear-gradient( rgba(R, G, B, A) X% , rgba(R, G, B, A) X% );
}

Box Transition Glow

Use transition and box shadows to flow on hover

.code-card .card-header {
    boder-radius: 8px;
    transition: all Xs ease-in-out;
}
.code-card:hover, .code-card:hover .card-header {
    box-shadow: inset Xpx Xpx Xpx rgba(R, G, B, A), Y Y Xpx rgba(R, G, B, A);
}
